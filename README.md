# MSF
Multi Simple Force Tools

#### ⇨  Install Script Di Termux
`###
$ termux-setup-storage  
$ pkg update && upgrade  
$ pkg install python  
$ pkg install git  
$ pip install bs4  
$ pip install requests  
$ pip install mechanize  
$ pip install futures  
$ pip install ipaddress
$ git clone https://github.com/Dicky-XD/MSF  
```
#### ⇨  Run Script
```
$ cd MSF  
$ python MSF.py 
```
